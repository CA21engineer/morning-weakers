# morning_weakers

![CI](https://img.shields.io/badge/build-passing-brightgreen)

## 概要

ハッカソンを手間なく企画・運営出来るスマートフォンアプリ

## 開発背景

これは2020/06/07(日)~2020/06/14(日)にCyberAgentの21卒の内定者内で開催された第1回ハッカソンの成果物です。
このレポジトリはFlutterチーム「朝弱いけん」のものです。
